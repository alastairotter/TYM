This is the start of the Track Your Mayor admin interface.

Some of the functionality is not yet working.

What is and isn't working:

  • Under the List menu item you can view the various data fields including mayors, municipalities, parties, and promises
  • In the List View you can sort the columns by clicking on the column headers.
  • A basic filter is available in List View. Type to automatically filter.
  • You can add new elements from the List View or via the Add menu itme.
  • Existing records can't be edited yet. This will be added shortly.
  • The Test Home page is just a sample of various ways we can pull out the data and display it. Still very rough.
