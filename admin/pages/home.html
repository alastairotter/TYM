This is the start of the Track Your Mayor admin interface.

Some of the functionality is not yet working.

What is and isn't working:

  • Under the List menu item you can view the various data fields including mayors, municipalities, parties, and soon promises
  • The add/edit/update functions are being tested on the mayors List View. Other list items may not have all the functionality yet.
  • In the List View you can sort the columns by clicking on the column headers.
  • A basic filter is available in List View. Type to automatically filter.
  • You can also add data on the List View. A separate Add menu item can be added to streamline the interface.
  • The option to edit fields doesn't work yet. It's next on the agenda.
